# CatchIT: Integrated Task and Rollout Management System
---
## Overview
CatchIT is a centralized web-based project management platform designed for the Integrated Marketing Communications (IMC) Department of CATCH2T28, the Batch Student Government representing CCS ID 124 students of De La Salle University.
The system streamlines project management by providing a single platform for project requests, task assignments, project tracking, communication, and document management. It aims to reduce overlooked tasks, improve workflow efficiency, and centralize project-related information.
---
## Features
### User Management
* User Registration
* Secure Login
* Committee and Position Assignment
### Project Management
* Project Request Submission
* Rollout Form Management
* Project Status Tracking
* Dashboard Overview
### Task Management
* Task Assignment
* Deadline Monitoring
* Progress Tracking
### Collaboration
* Project Discussion and Updates
* Notifications and Reminders
* Revision and Update Logs
### File Management
* Centralized File Repository
* File Upload and Storage
### Analytics
* Dashboard Analytics
* Performance Reports
* Workload Distribution Insights
### Search and Filtering
* Search projects by:
   - Committee
   - Project Name
   - Assigned Members
   - Deadlines
   - Status
