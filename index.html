About the System

Built for IMC.
Designed for clarity.

The IMC Rollout and Project Tracking System is designed to streamline project management within the Integrated Marketing Communications Department. It centralizes project requests, task assignments, status tracking, and communication to improve efficiency and reduce overlooked tasks.

From initial request submission to final deliverable, every stage of a project lives in one place — giving your team the visibility and coordination it needs to execute campaigns effectively.

Features

Everything your team needs

Purpose-built tools to keep IMC projects organized, on track, and fully transparent from start to finish.

Project Request Submission

Submit new project requests through a structured form that captures all the necessary information for the IMC team to begin work efficiently.

Rollout Form Management

Manage and organize rollout forms with version control, ensuring every campaign plan is documented and accessible in one central location.

Project Dashboard

Get a real-time overview of all active, pending, and completed projects at a glance — so nothing slips through the cracks.

Task Assignment

Assign specific tasks to team members, set deadlines, and track individual responsibilities within every project effortlessly.

Project Status Tracker

Monitor the live status of every project — from In Progress to Under Review to Complete — with a clear, visual progress indicator.

Project Discussion & Updates

Keep all project-related communication in context. Post updates, leave comments, and discuss changes directly on each project thread.

Notifications & Reminders

Receive automated alerts for task deadlines, status changes, and new assignments so your team stays on top of every update.

File Upload & Repository

Attach and organize all project-related files — briefs, designs, references — in a structured repository linked to each project.
